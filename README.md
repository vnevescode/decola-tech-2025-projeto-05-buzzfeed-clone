# 🧠 Buzzfeed Clone - Angular Quizz App

Este é um projeto clone inspirado nos quizzes interativos do Buzzfeed, desenvolvido com Angular 14. O app simula diferentes tipos de quizzes divertidos (como “Qual pet combina com você?” ou “Você seria herói ou vilão?”), com perguntas aleatórias, efeitos de carregamento e um design moderno com emojis e tons pastéis.  

---


## 🚀 Funcionalidades

- ⚡ Sorteio automático de quizzes ao carregar o app
- ✨ Estilo visual moderno com cores em tons pastéis + emojis 😄
- 🧠 Tela de loading inicial e tela de cálculo do resultado simuladas (2 segundos)
- 🔄 Botão “Refazer Quiz” para reiniciar com um novo quiz aleatório
- 📁 Suporte para múltiplos arquivos `.json` de quizzes (8 no total)
- 🔀 Perguntas aleatórias e resultados personalizados de acordo com as respostas

## 🛠️ Tecnologias utilizadas

---

## 📁 Estrutura dos Quizzes

Os arquivos `.json` com perguntas e resultados estão localizados na pasta:

```bash
src/assets/data/
```

Exemplos:
- `quizz_heroi_vilao.json`
- `quizz_pets.json`
- `quizz_comidas.json`
- `quizz_celebrities.json`
- `quizz_signos.json`
- `quizz_viagens.json`
- `quizz_estilos.json`
- `quizz_tecnologia.json`

## ▶️ Como rodar o projeto localmente

1. **Clone o repositório**
```bash
   git clone https://github.com/seu-usuario/seu-repositorio.git
 ```
2. **Instale as dependências**
```bash
   npm install
```
3. **Rode o projeto**
```bash
   ng serve
```
4. **Abra no navegador**
```bash
   http://localhost:4200
```
